# EduConf

This is a team educational project - a website for an educational conference built with Vite and vanilla JavaScript.

## Project Structure

- `src/`: Source files including HTML, JavaScript, and CSS
- `assets/`: Static assets
- `img-project/`: Project images organized by sections
- `partials/`: HTML partials for different sections

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

To start the development server:
```bash
npm run dev
```

To build for production:
```bash
npm run build
```

To preview the production build:
```bash
npm run preview
```

## Technologies Used

- Vite
- Vanilla JavaScript
- HTML5
- CSS3
